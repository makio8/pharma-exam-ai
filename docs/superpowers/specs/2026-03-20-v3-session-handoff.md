## 発見された5つの課題カテゴリ

### 課題1: 画像の余白が大きい（🔴 高優先・工数小）

**現象**: ページ末尾の問題がページ下端までクロップされ、巨大な余白・ページ番号（「−31−」等）・トンボが含まれる
**該当**: 103回問91、107回問197、100回問233
**原因**: `calcCropRegion` でページ末尾の問題は `imageHeight - 10` までクロップ
**解決策**:
```
sharp(image).trim({ threshold: 10 })  // 白い余白を自動除去
+ ページ番号領域（下端80px）のカット
```
**工数**: 30分

### 課題2: choices空だがテキスト選択肢で解ける問題がある（🔴 高優先・工数中）

**現象**: 表形式・リスト形式の選択肢がPDFから抽出できず、choices空+画像表示になっている
**該当例**:
- 103回問91: 表「分子間相互作用 → 現象」→ choicesに復元可能
- 100回問233: 表「植物 → 有毒成分」→ choicesに復元可能
- 100回問239: テキスト選択肢 → choicesに復元可能
**推定対象数**: 867問中 200-300問がchoices復元可能（残りは構造式・グラフ等で画像のまま）
**解決策**: AI Vision（Claude API or Gemini Flash）で画像から選択肢テキストを抽出
```
プロンプト例:
「この問題画像から選択肢をJSON形式で抽出してください。
選択肢がテキスト化可能な場合: {"extractable": true, "choices": [{"key": 1, "text": "..."}, ...]}
構造式・図の場合: {"extractable": false, "reason": "構造式"}」
```
**工数**: 2-3時間

### 課題3: question_textのOCR品質が低い（🟡 中優先）

**現象**: pdftotext -layout 由来の改行崩れ、下付き数字混入、不自然な文分断
**該当例**:
- 107回問197: 「₁₇歳男性。身長₁₇₅cm」（数字混入）
- 100回問8: 「1つ選」と「べ。」が分断
- 100回問2: グラフラベル「A B」が残る
**解決策**: 課題2と同時にAI Visionでquestion_textも再生成
**工数**: 課題2に含まれる

### 課題4: 連問（実践問題）の構造化（🟡 中優先・工数大）

**現象**: 実践の連問（問196-197、問232-233、問238-239等）が個別問題として表示。共通シナリオ・処方情報が重複
**エメリーの理想形**: 共通シナリオ+処方画像 → 子問題1 → 子問題2
**解決策**:
```typescript
// Question型に追加
linked_group?: string      // 例: "r107-196-197"
linked_scenario?: string   // 共通シナリオテキスト
linked_image_url?: string  // 共通の処方画像等
```
- PDFの「問 196−197」パターンから連問グループを自動検出
- UIで連問グループを一括表示
**工数**: 3-4時間（データ構造変更+UI変更+全年度の連問検出）

### 課題5: choicesあり問題の補助画像欠如（🟡 中優先）

**現象**: choicesありの問題でも処方箋・検査値表の画像が必要なケースがある
**該当**: 110回問286-287（処方情報がテキストだが画像の方が見やすい）
**解決策**: 課題4の連問構造化と同時に対応
**工数**: 課題4に含まれる

## 技術メモ

### pdftotext の限界（v2で学んだこと）
- `-layout` オプションは改行位置がPDF内部の文字配置に依存し、不安定
- `-bbox` は座標は正確だが、テキスト結合ロジックが必要
- 表形式のデータは抽出困難（セル構造が失われる）
- 数字が下付き・上付きとして埋め込まれている場合、位置がずれる

### AI Vision アプローチ（v3で採用予定）
- **メリット**: 画像から正確なテキスト抽出、選択肢のテキスト化判定が可能
- **コスト**: Claude API（@anthropic-ai/sdk 既存）or Gemini Flash（未設定）
- **レート制限**: Gemini無料枠20req/日。Claude APIは有料だが高品質
- **バッチ設計**: 867問全件ではなく、choices空の問題のみ対象。構造式問題はスキップ

### 既存インフラ
- `@anthropic-ai/sdk` は package.json にインストール済み
- `.env.local` に ANTHROPIC_API_KEY が設定済み（要確認）
- GEMINI_API_KEY は未設定
- PDFファイル: `/tmp/claude/q{year}-{type}.pdf`（全11年分存在）
- ページ画像: `/tmp/claude/exam-pages/{100-110}/`（全年分あり）
- sharp: インストール済み

### 画像問題の分類（867問の内訳予想）
| カテゴリ | 推定数 | 対応方針 |
|---------|--------|---------|
| 構造式・化学式 | ~200問 | 画像+番号ボタン維持（現状OK） |
| グラフ・図・フロー図 | ~150問 | 画像+番号ボタン維持 |
| テキスト表・リスト | ~200-300問 | AI Visionでchoices復元 |
| 処方箋・検査値表 | ~100問 | 連問の補助画像として構造化 |
| その他 | ~100問 | 個別判断 |
