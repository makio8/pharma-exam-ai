# Competitive Analysis: pharma-exam-ai（国試ノート）

**Date**: 2026-03-23
**Analyzed**: 直接競合6社 + 間接競合3社

---

## Market Overview

薬剤師国家試験対策アプリ市場は、2023年のEmeryリリースまで「実用的なアプリが存在しなかった」（パスメド記事）未成熟市場。Emeryが唯一の本格プレイヤーとして急成長（4.8星/437件）しているが、**UX・解説品質・信頼性に明確な弱点**がある。他アプリは問題数不足・UI老朽化・機能不足で競争力が低い。市場はまだ「1強+低品質多数」の構図であり、**高品質な2番手が参入する絶好のタイミング**。

青本（紙の参考書）が依然として学習の中心であり、アプリは「2周目以降の繰り返し演習」として併用されるポジション。つまり**紙の代替ではなく補完ツール**。

## Differentiation Strategy（差別化戦略）

### 🎯 勝ち筋: 「考えなくていい高機能」

エメリーの弱点は「高機能だがUXがツール型」。SEEDの弱点は「UXは良いが低機能」。
**この2つを統合したポジション = 「考えなくていい高機能」が空いている。**

| 戦略 | 具体策 |
|------|--------|
| **Double down（攻める）** | AI解説の品質、今日のメニュー、単元マスター、オフライン |
| **Close the gap（追いつく）** | CBTモード、模試モード（タイマー付き） |
| **Ignore（無視する）** | ランキング（初期段階では不要。ユーザー数が少ないとかえって逆効果）、手書きメモ（バグの温床。テキストメモで十分） |

### 差別化の3本柱

#### 柱1: 「開いたら迷わない」UX（vs Emeryの検索ツール型）

```
Emery: ホーム → 検索 → フィルター設定 → 問題選択 → 開始（5ステップ）
国試ノート: ホーム → 今日のメニュー → タップ → 開始（2ステップ）
```

これが**最大の差別化ポイント**。Emeryとの機能差ではなく、体験の質で勝つ。

#### 柱2: AI解説の読みやすさ（vs Emeryの改行なし解説）

Emeryレビューで最も多い不満。pharma-exam-aiのAI構造化解説は：
- 改行・見出し付き
- 重要ポイントの強調
- 関連知識のリンク

**体験してすぐに違いがわかるポイント** → 口コミの起点になる

#### 柱3: データが消えない安心感（vs Emeryのデータ消失）

「アプデで書き込み全消失」はEmery最大のトラウマ。
PWA + ローカルストレージ優先設計は、地味だが**信頼の基盤**。

## Competitive Threats（警戒すべき脅威）

| 脅威 | 深刻度 | 対応策 |
|------|--------|--------|
| **Emeryがオフライン対応する** | 🟡 中 | 先にUX品質で差をつけておく |
| **Emeryが解説品質を改善する** | 🟡 中 | AI解説はコスト的に人力より速く更新可能 |
| **e-RECがアプリ化する** | 🔴 高 | 動画解説+無料は強力。UXと暗記機能で差別化 |
| **薬ゼミが本格参入する** | 🔴 高 | ブランド力が圧倒的。独自AIコンテンツで対抗 |
| **新規AIネイティブアプリが登場** | 🟡 中 | 先行してデータ+コンテンツを蓄積しておく |

## 攻めるべきセグメントと順序

### Phase 1: 「効率型」の6年生を獲得（今〜3ヶ月）

**なぜ最初か**: 市場の60%を占め、口コミ伝播力が最大（同級生に広める）

| 施策 | 目的 |
|------|------|
| 演習ページUXリデザイン | Emeryからの乗り換え動機 |
| スマートプリセット実装 | 「2タップで演習開始」体験 |
| AI解説の品質訴求 | 体験後すぐに違いがわかる差別化 |

### Phase 2: 「入門型」の4年生を独占（3〜6ヶ月）

**なぜ2番目か**: 競合が完全に無視している空白市場。一度獲得すれば2年間使う

| 施策 | 目的 |
|------|------|
| CBTモード追加 | 必須問題+基礎科目に限定した簡易UI |
| オンボーディング分岐 | 「4年生CBT」「6年生国試」「既卒」を選択 |
| 正答率80%以上フィルター | 「まず基本を全部○に」というニーズ |

### Phase 3: 「分析型」の既卒受験者で収益化（6ヶ月〜）

**なぜ3番目か**: 課金意欲最大だが、母数が少ない。口コミ基盤ができてから

| 施策 | 目的 |
|------|------|
| 模試モード（タイマー+本番配分） | 本番シミュレーション |
| AI弱点分析レポート | 「あなたの不合格パターンはここ」 |
| プレミアムプラン（月500-800円） | Emeryより安く、質で勝つ |
