# pharma-exam-ai 次セッション要件書

## 🔴 最優先: 全問データ品質監査

### 発見された問題（2026-03-24）
実機で演習中に、画像以外のデータ品質問題が多数発見された：

1. **選択肢テキストの分断**: 「γ-アミノ酪酸GABAA受容体」が「γ」だけに切れる
2. **選択肢→問題文への混入**: 選択肢の後半部分が question_text に入り込む
3. **問題文→選択肢の分離不良**: 複合選択肢（「K+ チャネル活性化」等）が分断
4. **シナリオ→問題文への混入**: 連問でシナリオが question_text に入る（Tier1レビューで既知）

### 影響範囲
- Tier1の78問だけでなく、**全4000問に影響する可能性**
- 特に薬理（受容体名+サブタイプ）、生物（遺伝子名）など、特殊文字を含む選択肢で多発

### 対応方針（GPT-5.4推奨）

| 順 | 施策 | 内容 | 工数 |
|:--:|------|------|------|
| **1** | **全問バリデーター作成** | 選択肢の異常を自動検出するスクリプト | 2-3h |
| **2** | **e-REC/PDF差分照合** | 元データとの不整合を問単位で可視化 | 2-3h |
| **3** | **ルールベース一括修復** | パターン化できる問題を自動修正 | 1-2h |
| **4** | **レビューUI** | 残件を人手で確認・修正 | 2-3h |

### バリデーターで検出すべき異常パターン
```
- 選択肢テキストが極端に短い（3文字以下）
- 問題文に「受容体」「チャネル」等が連続で並ぶ（選択肢混入の兆候）
- 選択肢が全て同じ長さ（分断パターン）
- linked_scenario が question_text と高類似
- choices が隣接問題と高類似（紐付けエラー）
- image_url 設定済みだが画像ファイルが存在しない
```
