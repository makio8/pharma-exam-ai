## 知見・注意事項

| 項目 | 詳細 |
|------|------|
| **同一JSONへの並列書き込み** | 競合リスクあり。各エージェントは別年度のファイルに書くこと |
| **JSONエスケープ漏れ** | ダブルクォートを含む解説文でJSON壊れる。生成後にJSON.parseで検証推奨 |
| **テンプレート優先マージ** | merge-explanations.tsは【ポイント】を含む解説を常に優先する |
| **e-RECの実践問題制限** | 問196-345の約135問はe-RECにHTMLなし（連問形式）。正答のみ存在 |
| **merge-explanations.ts** | years配列は [100-110] に更新済み |
